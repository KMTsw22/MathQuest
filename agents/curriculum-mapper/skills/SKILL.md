# Skill: Common Core Math Standards 조회

## CCSS 전체 표준 구조 참조

Common Core 수학 표준은 학년(K–8) 및 고등(HS)별로 공식 문서에 정의되어 있다.

### 표준 조회 방법
1. 학년과 영역(Domain)으로 해당 클러스터(Cluster) 식별
2. 클러스터 내 개별 표준(Standard) 번호 확인
3. 표준 설명문에서 학습 목표 추출
