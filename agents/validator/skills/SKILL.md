# Skill: 수학 계산 검증 방법

## 1. 분수 계산 검증

### 덧셈/뺄셈
```
a/b + c/d = (a×d + c×b) / (b×d) → 기약분수로 약분
예: 1/3 + 1/4 = (4 + 3) / 12 = 7/12  ✓

검증: answer = "7/12" → 맞음
검증: answer = "2/7" → 틀림 (wrong_answer 반려)
```

### 곱셈
```
a/b × c/d = (a×c) / (b×d) → 기약분수로 약분
예: 2/3 × 3/4 = 6/12 = 1/2  ✓
```

### 나눗셈
```
a/b ÷ c/d = a/b × d/c = (a×d) / (b×c)
예: 3/4 ÷ 1/2 = 3/4 × 2/1 = 6/4 = 3/2  ✓
```

### 기약분수 확인
```
GCD(분자, 분모) = 1이면 기약분수
예: 4/6 → GCD(4,6)=2 → 기약분수 아님 → 2/3이 정답
```

## 2. 정수 / 큰 수 계산 검증

### 곱셈 검증 (격자법)
```
47 × 23:
  40 × 20 = 800
  40 × 3  = 120
   7 × 20 = 140
   7 × 3  =  21
         = 1081  ✓
```

### 나눗셈 검증 (역산)
```
몫 × 제수 + 나머지 = 피제수
예: 47 ÷ 5 = 9 나머지 2
검증: 9 × 5 + 2 = 47  ✓
```

## 3. 방정식 검증

### 일차방정식
```
풀이 결과(x값)를 원래 식에 대입하여 성립하는지 확인
예: 2x + 3 = 11 → x = 4
검증: 2(4) + 3 = 11 ✓
```

### 비례식
```
a/b = c/d → a×d = b×c (교차 곱)
예: 3/4 = 6/8 → 3×8 = 24, 4×6 = 24 ✓
```

## 4. 기하 계산 검증

### 넓이
```
직사각형: 가로 × 세로
삼각형: (밑변 × 높이) / 2
원: π × r²  (π ≈ 3.14 또는 22/7 사용 명시 필요)
```

### 부피
```
직육면체: 가로 × 세로 × 높이
원기둥: π × r² × h
```

### 피타고라스
```
a² + b² = c²
검증: c² - a² = b² 성립 확인
```

---

## 5. 통계 검증

### 평균
```
(모든 값의 합) / (개수)
검증: 평균 × 개수 = 합계
```

### 최빈값/중앙값
```
중앙값: 정렬 후 가운데 값 (짝수 개면 중간 두 값의 평균)
최빈값: 가장 많이 나오는 값
```

---

## 6. 선택형 문제 검증 절차

1. 문제 문장에서 정답 직접 계산
2. A–D 보기 중 계산 결과와 일치하는 항목 찾기
3. `answer` 필드의 선택지(예: "C. 1/2")와 대조
4. 일치 → 승인 / 불일치 → wrong_answer 반려
